# Metric Unit Converter

A simple and intuitive web-based metric unit converter supporting metric and imperial measurements.

## Features

- Convert between 8 different length units:
  - Metric: mm, cm, m, km
  - Imperial: inch, foot, yard, mile
- Real-time conversion calculations
- View all unit conversions at once
- Responsive design
- Input validation

## Supported Units

| Unit | Abbreviation |
|------|--------------|
| Millimeter | mm |
| Centimeter | cm |
| Meter | m |
| Kilometer | km |
| Inch | inch |
| Foot | foot |
| Yard | yard |
| Mile | mile |

## How to Use

1. Open `index.html` in a web browser
2. Enter a value to convert
3. Select the source unit (From)
4. Select the target unit (To)
5. Click "Convert" or press Enter
6. View the result and all possible unit conversions

## Files

- `index.html` - Main HTML interface
- `style.css` - Styling
- `app.js` - Application logic with converter

## Examples

- 1 meter = 100 cm
- 1 inch = 2.54 cm
- 5 miles = 8.047 km
- 1 foot = 304.8 mm
